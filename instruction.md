# Project Overview

Хочу написать телеграм бота на pyton. 
Бот можно будет добавлять в группы.
Если в группу добавиться новый участник, бот должен будет сохранить его telegram_id и ник.
Если написать боту команду /134950 - то он должен показать список всех пользователей, которых он когда то сохранил
